### 📂 Структура проекта
- `test/` — файлы с автотестами и конфигурацией (`test_ui.py`, `test_api.py`, `config.py`).
- `pytest.ini` — конфигурация pytest и регистрация маркеров (ui, api).
- `README.md` — описание проекта и подробные инструкции по запуску.
- `requirements.txt` — список необходимых библиотек и зависимостей.
- `allure-results/` — папка с результатами прогонов для генерации отчетов.(генерируется локально после запуска).


### 🛠 Стек технологий
- **Python 3.11**
- **Pytest** — тестовый фреймворк.
- **Selenium** — автоматизация веб-интерфейса.
- **Requests** — тестирование программного интерфейса (API).
- **Allure** — построение наглядных и детальных отчетов.

### 🚀 Инструкция по запуску
1. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt

2. Запуск тестов:
   - `pytest -m ui` — только UI тесты
   - `pytest -m api` — только API тесты
   - `pytest` — все тесты вместе

3. Сформируйте отчет Allure:
Соберите данные: pytest --alluredir=allure-results
Откройте отчет: allure serve allure-results
